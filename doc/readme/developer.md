## Developer

### Test

To run the test suite:

```
npm test
```

### Cover

To generate code coverage:

```
npm run cover
```

### Docs

To build all documentation:

```
npm run docs
```

### Manual

To build all man pages (requires [manpage][]):

```
npm run manual
```

### Readme

To build the readme file from the partial definitions (requires [mdp][]):

```
npm run readme
```
